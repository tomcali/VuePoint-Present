The Client:

Research Publishers LLC

Publish books and research case studies

Two major components to project:

  1. Reconstruct existing simple static website into a single page application

  2. Create a database-driven content management system for employees




Application Logic

Buyer Website

  • Definition of components and routes
  • vue.js, vue-router.js, implemented via node.js, npm, and webpack
  • Buyer visits homepage
  • Home.vue
  • User.vue
  • Publications and Client Services
  • Request a product or service
  • Checkout by submitting the request

Application Logic

Employee Data Entry Website

  • Assume employee's skillset is basic data entry with no coding background
  • Create/update a book
  • Create/update an author or editor
  • Create/update a cover art artist
  • Submit changes to database
  • SQLite as a transactional SQL database engine
  • Sequelize

Technologies Used

Technologies Used

  1. Back-end technologies used to host the content management system include:
    • JavaScript packages, including Node, Express
    • SQLite3 Database
    • Sequelize
  2. Front-end technologies used to host the customer website and content management system:
    • Vue.js including templating tools
  3. Project Development tools:
    • WebStorm
    • GitHub (ZenHub Project Management)
    • Web-browser-based presentation on GitHub
